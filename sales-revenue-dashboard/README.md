## Enterprise Sales & Revenue Intelligence Dashboard

### Overview
Developed an interactive Power BI dashboard to monitor revenue performance, regional trends, and key business KPIs for leadership decision-making.

### Key Features
- Revenue and profit trend analysis
- Region-wise performance breakdown
- KPI tracking (Revenue, Growth %, Targets)
- Drill-down insights for detailed analysis

### Tools & Technologies
Power BI, SQL (CTEs, Joins, Window Functions), Python (Pandas)

### Key Impact
- Improved reporting efficiency by 40%
- Enhanced decision-making with real-time insights
- Optimized data models using star schema
